## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Configure the iOS App (only MacOS)

   ```bash
   npx expo prebuild --platform ios
   ```

3. Start the app (only MacOS)

   ```bash
   npx expo run:ios
   ```
